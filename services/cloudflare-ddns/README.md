# Cloudflare DDNS

A tiny DDNS service that will sync your WAN IP with CloudFlare DNS records.

|-----------------|-------------|---------------|
| `CF_API_KEY`    |             | _N/A_ |
| `CF_ZONE_ID`    |             | _N/A_ |
| `CF_RECORD_ID`  |             | _N/A_ |
